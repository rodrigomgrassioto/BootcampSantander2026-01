# Santander Bootcamp 2026.1 - Trilha Java 🚀

Repositório criado para centralizar todos os desafios de código, exercícios e projetos práticos desenvolvidos durante o **Santander Bootcamp 2026.1**, programa intensivo de aceleração em tecnologia para o qual tive a grande satisfação de ser selecionado.

O objetivo deste espaço é registrar minha evolução técnica, documentar o aprendizado prático e demonstrar a aplicação de conceitos modernos de engenharia de software utilizando a linguagem Java.

---

## 🛠️ Estrutura do Repositório

Os projetos estão organizados por módulos evolutivos, acompanhando o cronograma oficial do bootcamp:

*   **[Exercício 1: Fundamentos da Linguagem Java](./Exercicios/1%20-%20Fundamentos%20da%20Linguagem%20de%20Programa%C3%A7%C3%A3o%20Java/)**
    *   Consolidação de sintaxe básica, declaração de variáveis e tipos primitivos.
    *   Domínio de operadores lógicos, aritméticos, precedência e curto-circuito.
    *   *Nota: Contém as 4 atividades iniciais centralizadas em arquivos de execução direta.*

*   **[Exercício 2: Estruturas de Controle em Java](./Exercicios/2%20-%20Estruturas%20de%20Controle%20em%20Java/)**
    *   Fluxos de tomada de decisão com estruturas condicionais avançadas (`if/else`).
    *   Uso prático de *Switch Expressions* modernos (`->`) e controle de blocos com `yield`.
    *   Estruturas de repetição eficientes (`for`, `while`, `do-while`) e controle de fluxo com `break` e `continue`.

*   **[Exercício 3: Java e a Arte da Abstração com Classes e Encapsulamento](./Exercicios/3%20-%20Java%20e%20a%20Arte%20da%20Abstra%C3%A7%C3%A3o%20com%20Classes%20e%20Encapsulamento/)**
    *   Aplicação dos pilares de Programação Orientada a Objetos (POO).
    *   Desenvolvimento de projetos individuais e isolados com múltiplas classes corporativas.
    *   **Projeto em Destaque:** `3 Máquina de banhos em pet` - Sistema modelado para gerenciamento de estados, regras de negócio de limpeza e ocupação de animais.
	
*	**[Exercício 4: Herança e Polimorfismo Avançado em Java](Exercicios/4%20-%20Heran%C3%A7a%20e%20Polimorfismo%20em%20Java/)**  
     *   Aprofundamento na criação de arquiteturas extensíveis, reutilização de código e polimorfismo dinâmico.
     *   Ticket (Cinema Ticket System):** Hierarquia de precificação dinâmica para bilheteria de cinema, aplicando o padrão criacional *Static Factory Method* para reaproveitamento limpo de objetos sem duplicação de strings (Princípio DRY).
     *   RodrigoUsersControl (User Access Control System):** Sistema de controle de acessos corporativos (RBAC). Uso de classes abstratas e injeção de estado em cascata para blindagem rígida de permissões administrativas (`isAdmin`), associado a validações defensivas de segurança para alteração de credenciais.
     *   RodrigoClock (World Clock Architecture):** Sistema de sincronização e regionalização de fusos horários (24h vs. 12h AM/PM) utilizando recursos de última geração como *Sealed Classes* (Java 17) para herança restrita e *Pattern Matching para estruturas Switch* (Java 21).

---

## 🚀 Tecnologias e Metodologias Aplicadas

*   **Java (Versões Modernas)**: Utilização de recursos recentes da linguagem para escrita de código limpo.
*   **Programação Orientada a Objetos (POO)**: Abstração, encapsulamento e isolamento de responsabilidades.
*   **PBL (Project-Based Learning)**: Aprendizado baseado na resolução de problemas práticos e cenários do mundo real.
*   **Pensamento Computacional**: Resolução de desafios focando em decomposição, reconhecimento de padrões, abstração e algoritmos.

## 🧑‍💻 Como Executar os Projetos

Cada pasta possui sua própria estrutura de código fonte (`/src`). Para rodar qualquer um dos desafios:
1. Abra a subpasta do projeto desejado na sua IDE de preferência (IntelliJ IDEA, VS Code, etc.).
2. Certifique-se de ter o JDK (Java Development Kit) instalado e configurado.
3. Execute o arquivo `Main.java` correspondente.
